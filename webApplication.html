Web applications

  • The application which required internet and browser is refered as "Web applications"
  • To develop this type of applications programmers are using "Web technologies"(or)"advance java"
  • Web applications does not need installation in client machine(laptop/desktop)
  • For example
  1. Amazon
  2. Facebook
  3. Flipkart
  4. udemy

Note:

  • Browser is a standalone application which is used by users to access web applications with the help of internet
  • Each and every browser requireds installation in client machine
  • For example
  1. chrome
  2. fireox
  3. opera
  4. microsoft edge

Multimedia tags
  1. Programmer prefer multimedia tags in order to provide advertisement on the browser
  2. There are three multimedia tags are as follows:-
  • Image
  • Video
  • Audio
  1. img is used to provide images on the browser. It takes four attributes are as following:-
  • src:- it stands for source attribute which is bascially used to provide image filename/path
  • width&height:- they are used to control the dimension of an image
  • alt:-it is stands for "alternate text" used to specify some kind of message if given filename does not exists
  • video&audio is used to specify video files and audio files on the browser. The common attribute between tags are "src&controls where controls is used to display pause,play,buttons for the audio&videos.
  • To play video automatically on webpage make use of "autoplay" attribute with "muted"

    • Note
    1. As a programmers we can change dimension of video and image by using width&height attribute but it's not best practice
    2. If programmers are using autoplay&muted attribute then controls attribute is optional
    3. Always autoplay must&should use with muted
    4. If image and 3 file is present at different location then for scr attribute pass the "path/location of an image"

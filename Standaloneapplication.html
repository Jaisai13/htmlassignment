Standalone application

  • The application which does not required internet is refered as "Standalone application"
  • To develop this types of application programmers are using "core java"
  • Each and every user must install standalone application in their laptop(or)desktop[client machine].
  • To run Standalone application programmers required windows,linux,unix,mac operating systems.
  • For example
  1. Editplus
  2. eclipse
  3. notepad
  4. ms-office
